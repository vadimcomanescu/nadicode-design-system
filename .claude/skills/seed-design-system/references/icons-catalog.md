# Icons Catalog

73 animated icons with real SVG path animations on hover.
Import from `@/components/ui/icons`.

---

## Usage

```tsx
import { SettingsIcon, BellIcon, SearchIcon } from "@/components/ui/icons"

// Size in pixels, default 28
<SettingsIcon size={16} />
<BellIcon size={20} />
<SearchIcon size={24} />
```

For icons **not** in this catalog, use `lucide-react` directly:
```tsx
import { SomeOtherIcon } from "lucide-react"
<SomeOtherIcon size={16} />
```
